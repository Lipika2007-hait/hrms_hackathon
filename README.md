# hrms_hackathon
A web-based Human Resource Management System built to simplify employee and HR managemet tasks.

### Human Resource Management System(HRMS)

## Project Overview:
The Human Resource Management System(HRMS) is a web-based application developed to simplify and manage various human resource activities within an organization. It provides seperate login access for HR and Employees, allowing efficient management of employee records, attendance, leave requests, and personal profiles.
The project is developed as part of a hackathon using PHP and MySQL.

## objectives:

1. Digitize employee management.
2. Reduce manual paperwork.
3. Provide secure login for HR and Employees.
4. Manage employee information efficiently.
5. Improve communication between HR and employees.

## Features:

1. HR Login
2. Employee Login
3. Employee Registration
4. Secure Authentication
5. HR Dashboard
6. Employee Dashboard
7. Attendance Management
8. Leave Management
9. Employee Profile Management
10. View Employee Records
11. Update Employee Details
